# Architecture Standardisation Policy

All modules must conform to:
- Minimum Architecture Template  
- Naming Conventions  
- Folder Structure  
- Versioning Rules  
- QA Requirements  
- QA-of-QA Requirements  

Maturion Foreman must:
1. Validate existing modules.
2. Identify inconsistencies.
3. Request missing files.
4. Create a standardisation plan.
5. Mark module as compliant or non-compliant.
