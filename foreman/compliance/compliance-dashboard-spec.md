# Compliance Dashboard Specification

## Purpose
Display compliance posture in real time.

## Metrics
- Compliance score (per module)
- Control violations count
- Severity of violations
- Compliance trends over time
- Required remediation steps
- Tenant-level compliance score
- Platform-level compliance score

## Drill Down
- Control-specific failures
- Module-specific issues
- QA vs Compliance mismatch
- Evidence attachments
