# AI Performance Metrics Specification

## Purpose
Show how effectively Maturion operates.

## Metrics
### Per Tenant
- total AI calls
- total tokens used
- cost per tenant
- most-used models
- module-level usage

### Per Module
- token usage
- average call length
- AI dependencies
- cost drivers

### Per User
- usage patterns
- model selection
- error history
- AI maturity score

### Global
- total operational cost
- peak usage times
- slow call detection
- optimization recommendations
