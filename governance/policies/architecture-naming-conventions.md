# Naming Conventions

## Filename Pattern
`{MODULE}_{COMPONENT}_v{MAJOR}.{MINOR}.md`

Examples:
- ERM_EDGE_FUNCTIONS_v1.1.md
- THREAT_WATCHDOG_LOGIC_v1.0.md

## Versioning
- v0.X = draft  
- v1.X = production  
- v2.X = second generation
